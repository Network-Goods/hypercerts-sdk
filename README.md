# Hypercert SDK

## Set up

`yarn install`

`yarn lerna run bootstrap`

`yarn lerna run tsc`

## Packages

### Contracts

Typed instances of Hypercert protocol

### Metadata

Metadata validator/generator/uploader

### Graph

GraphQL client for Hypercerts with predefined queries and client

### Types

* Contracts
* Graph entities
* Hypercert domain
